# ClickerGame

Android-кликер с системой улучшений, достижениями и сменой визуального состояния овечки. Игрок собирает шерсть, открывает улучшения, а овечка постепенно изменяет внешний вид.

---

## Возможности приложения

- **Кликер** - нажатие на овцу приносит игровые очки (шерсть)
- **Система улучшений** - покупка инструментов увеличивает ценность клика:
  - Ржавые ножницы (500 очков) - клик = 3
  - Острые ножницы (2500 очков) - клик = 10
  - Электрическая машинка (3500 очков) - клик = 50
- **Достижения** - при достижении определённого количества очков открываются экраны достижений
- **Блокировка прогресса** - после накопления нужной суммы игра блокируется до покупки улучшения
- **Три вида изменения овечки** - автоматическая смена внешности после улучшений
- **Сохранение прогресса** - игра автоматически сохраняет состояние при каждом клике
- **Фоновая музыка** - возможность включения/выключения в настройках
- **Экраны правил и настроек** - полноэкранные Activity с анимацией кнопок

---

## Структура Activity

| Активность | Назначение |
|------------|------------|
| `SplashActivity` | Стартовый экран, загрузка сохранений |
| `PastureActivity0` | Обычная овечка (начальное состояние) |
| `PastureActivity2500` | Подстриженная овечка (при достижении 2500 очков) |
| `PastureActivity5000` | Короткостриженная овечка (при достижении 5000 очков) |
| `AchievementActivity500/2500/3500` | Экраны достижений |
| `ImprovementsActivity` | Магазин улучшений |
| `GameSettingsActivity` | Настройки игры (звук, сохранение, сброс) |
| `SettingsActivity` | Настройки на главном экране |
| `RulesActivity` | Правила игры |

---

## Навигация

| Действие | Результат |
|----------|-----------|
| Нажатие на овцу | +1 / +3 / +10 / +50 очков |
| Кнопка улучшений | Открытие магазина улучшений |
| Кнопка настроек | Открытие настроек игры |
| Кнопка правил | Открытие экрана с правилами |
